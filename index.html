About this site

This website is a timer for the Pomodoro time management method. In the Pomodoro technique you break down your work into intervals usually 25 minutes with 5 minutes of break in between and a longer 15 minutes break after 4 pomodoros.

How to use this site

Press the Start button to start the timer, once the timer reaches 0 it will switch to a short break, where you can press the Start button to start the break.
Under the settings you can change all three duration and the number of pomodoros before a long break.
You can change the background color of each section.
With the Skip button you can end a section early if you want.
